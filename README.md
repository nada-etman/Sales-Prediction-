# Sales Prediction using Python

## Overview

This project develops a machine learning model to predict product sales based on advertising investments across different marketing channels. The project demonstrates how predictive analytics can support data-driven marketing decisions.

## Dataset

The dataset contains advertising expenditure across multiple channels along with the corresponding sales values.

## Objectives

- Explore advertising data
- Clean and preprocess the dataset
- Train a regression model
- Predict future sales
- Evaluate model performance

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training
6. Sales Prediction
7. Model Evaluation

## Results

The model estimates future sales based on advertising expenditure and demonstrates the practical use of machine learning for business forecasting and marketing analysis.
